Insert counters to your site easily.

== Description ==

Insert counters to your site easily. After installing and activating plugin link Counter Sections appears to WP admin menu. From there user can manage counters.

Set of icons will be uploaded to media library on activation of the plugin (because of that, activation of the plugin will take little more time than ususally). Those icons can be used as images in counters.
